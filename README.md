# 梦境工厂React Native客户端
梦境工厂一直没有iOS客户端，趁现在用React Native做一版iOS，以后Android版本也会切过来。

## 用到的第三方库
 - [React Navigation](https://reactnavigation.org)
 - [react-native-scrollable-tab-view](https://github.com/skv-headless/react-native-scrollable-tab-view)
